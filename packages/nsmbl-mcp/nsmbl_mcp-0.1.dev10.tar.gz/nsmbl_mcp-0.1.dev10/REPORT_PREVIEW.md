## Color Scheme

### Primary Colors
- **Purple Gradient**: `#667eea â†’ #764ba2` (Header background)
- **Chart Line**: `#667eea` (Purple - equity curve)
- **Success Green**: `#10b981` (Positive returns)
- **Danger Red**: `#ef4444` (Drawdowns, negative returns)

### Neutral Colors
- **Background**: `#ffffff` (White)
- **Card Background**: `#f9fafb` (Light gray)
- **Text**: `#1f2937` (Dark gray)
- **Subtle Text**: `#6b7280` (Medium gray)
- **Borders**: `#e5e7eb` (Light gray)

## Typography

- **Font Family**: System fonts (-apple-system, BlinkMacSystemFont, Segoe UI)
- **Header Title**: 36px, Bold (700)
- **Metric Values**: 32px, Bold (700)
- **Chart Titles**: 20px, Semi-bold (600)
- **Labels**: 14px, Semi-bold (600), Uppercase

## Responsive Behavior

### Desktop (> 1024px)
- 3-column metric grid
- Full-width charts
- Max width: 1200px
- Centered on page

### Tablet (768px - 1024px)
- 2-column metric grid
- Charts adapt to width
- Readable spacing

### Mobile (< 768px)
- 1-column metric grid
- Charts stack vertically
- Touch-friendly interactions

## Interactive Features

### Hover Effects
1. **Metric Cards**: Lift up 4px with shadow
2. **Chart Lines**: Highlight nearest point
3. **Tooltips**: Show formatted values

### Chart Interactions
1. **Zoom**: Click and drag on chart
2. **Pan**: Drag to move view
3. **Reset**: Double-click to reset zoom
4. **Hover**: Unified crosshair across charts

## File Size & Performance

- **Typical Size**: 40-60KB
- **Load Time**: < 100ms
- **Render Time**: < 200ms
- **Interactive**: Smooth 60fps

## Why This Approach Works

1. **Zero Latency**: No code generation = instant visualization
2. **Professional**: Better than most custom LLM-generated charts
3. **Interactive**: Plotly provides rich UX (zoom, pan, hover)
4. **Shareable**: Single file, works anywhere
5. **Consistent**: Every report looks polished
6. **Maintainable**: Easy to enhance template later
