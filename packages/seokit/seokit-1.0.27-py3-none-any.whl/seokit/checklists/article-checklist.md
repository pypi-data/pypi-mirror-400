# Article Checklist
> Version: 1.0 | Updated: 2025-12-20

## Content Structure
- [ ] Compelling H1 with main keyword
- [ ] Hook in first paragraph
- [ ] Clear thesis statement
- [ ] Logical section progression
- [ ] Strong conclusion with CTA

## SEO Optimization
- [ ] Main keyword in first 100 words
- [ ] Keyword density 1-2%
- [ ] LSI keywords naturally included
- [ ] Internal linking opportunities marked
- [ ] External authority links planned

## Readability
- [ ] Short paragraphs (3-4 sentences max)
- [ ] Bullet points for lists (3+ items)
- [ ] Subheadings every 300 words
- [ ] Active voice preferred
- [ ] One idea per sentence

## E-E-A-T Compliance
- [ ] Author expertise demonstrated
- [ ] Sources cited where applicable
- [ ] Up-to-date information
- [ ] Balanced perspective
- [ ] Clear, accurate claims

## Engagement
- [ ] Questions to reader
- [ ] Examples and case studies
- [ ] Data/statistics with citations
- [ ] Actionable takeaways

## Technical
- [ ] Meta description drafted
- [ ] Word count meets target
- [ ] No duplicate content
- [ ] Grammar and spelling checked

## Quality Signals
- [ ] No word repetition within paragraphs
- [ ] Complete sentences (subject + predicate)
- [ ] Smooth transitions between sections
- [ ] Comparison content uses tables
