This is an interactive scatter plot developed at CERFACS in order to explore the internal structure of a code.

How to use it

Highlight a point and its path by hovering over it. Click on it to keep the selection on and access details information. If you hold Ctrl while clicking a node, it will open the corresponding code file in VSCode.

For more, keep a look at the full documentation.

Shortcuts

h : Hide a point

Shift + h : Reveal all hidden points

f : Flag/unflag a point

Shift + f : Unflag all points
